Multi-Threading

This function opens the door for implementation a concept of "parallel computing".
Simply pass 2 or more functions as arguments to the vkthread.runAll(). That's all! Function vkthread.runAll() is similar to vkthread.run(), but let you to start simultaneous execution of multiple functions in parallel threads. On multi-core CPU this can significantly increase application performance.
Function's argument is an array, each element of which is argument list (see vkthread.run() for details). 

function foo(start,end){
	var sum = 0,
		dummy = 'a,b,c,d,e',
	    ix;
		
	for(ix=start; ix<end; ix++){
		dummy.split(',').join('-'); // do something to slowdown the process
		sum = sum+ix;
	}
	return sum;
} 
vkthread.runAll([ [foo, [0, 4e6]], // first function
		  [foo, [4e6,8e6]] ] //second function
		).then(function (data) {
			var bar = data[0]+data[1];
			console.log(bar);
		});

vkthread.runAll() is a Promise object, so it depends on promise library. Plugin supports two libraries: jQuery and when.js (see "Documentation" for details.) There is no "Callback" version for this function.
