Promise-based API
Multiple Threads:   vkthread.runAll()

This function opens the door to implementation concept of "parallel computing". You simply pass 2 or more functions as arguments to the vkthread.runAll(), that's all! On multi-core CPU this can significantly increase application performance (see "Parallel Computing Demo" page for details.)

It is Promise object, so you need to include when.js library (see Documentation for details) 

function dummySum(start,end){
	var sum = 0,
	    ix,
	    dummy = 'a,b,c';
		
	for(var ix=start; ix<end; ix++){
		dummy.split(',').join('-');
		sum = sum+ix;
	}
	return sum;
} 
function runAll_dummySum(){

	vkthread.runAll([ [dummySum, [0,  3e6]], 
			  [dummySum, [3e6,7e6]] ]
			).then(function (data) {
		document.getElementById('demo_result')
				.innerHTML = data[0]+data[1];
		})
}
