Standalone Function

This page demonstrates advantage of using vkThread() plugin.

- Click "Run in Page" button. It runs lenghty havy process. UI will be frozen for a few seconds.
- Click "Run in Thread" button. It runs the same process, but now UI is not affected. 

function foo(str) {
	var ret, ix;
	for(ix=0;ix<1e7;ix++){
		ret = str.split(',').join('-');
	}
	return ret;
}
function run_foo(){
	document.getElementById('demo_result').innerHTML = foo('a,b,c');
}

function run_foo_in_thread(){
	vkthread.exec(foo,['x,y,z'], function(data){
		document.getElementById('demo_result_thread').innerHTML = data;
	});
}
