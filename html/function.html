Standalone Function

This page demonstrates advantage of using vkThread() plugin.

- Click "Run in Page" button. It runs lenghty havy process. UI will be frozen for a few seconds.
- Click "Run in Thread" button. It runs the same process, but now UI is not affected.

Here is the function we run: 
function foo(str) {
	var ret, ix;
	for(ix=0;ix<1e7;ix++){
		ret = str.split(',').join('-');
	}
	return ret;
}

var bar = foo('a,b,c');
console.log(bar);

/* Callback style */
vkthread.exec(foo,['a,b,c'], function(data){
	var bar = data;
	console.log(bar);
});

/* Promise style */	
vkthread.run(foo,['a,b,c']).then(function(data){
	var bar = data;
	console.log(bar);
});
