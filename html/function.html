Standalone Function

This page demonstrates advantage of using vkThread() plugin.

- Click "Run in Thread" button. It runs extensive process which takes a few seconds, but UI is not affected.

Here is the function we run: 
function foo(str) {
    var ret, ix;
    for(ix=0;ix<1e7;ix++){
	    ret = str.split(',').join('-');
    }
    return ret;
}



var param = {
	fn: foo,
	args: ['a,b,c']
};

vkthread.exec(param).then(function(data){
	console.log(data);
});
