Overview

vkThread is a javascript plugin, which allows you to execute any function of a your code in a thread.

Almost any type of function can be opened in a thread with this plugin:

  • Regular functions
  • Object's methods
  • Functions which dependents on library
  • Anonymous functions
  • Lambda functions

Working with this plugin you can use 2 type of coding styles:
  • "Callback" ( no dependencies in this case)
  • "Promise" (you need to link one of two promise libraries: either jQuery or when() from cujojs )

Here is a basic example: 

function sum(num1, num2) {
	return num1 + num2;
}
/* To run this function in a page */
var foo = sum(2, 3);
console.log(foo);

To run this function in a thread:

/* Callback style */
vkthread.exec(sum, //function to run in a thread;
             [2,3], //arguments for the function;
             function(data){ // callback function;
                var foo = data; 
                console.log(foo);
            });

/* Promise style */
vkthread.run(sum, //function to run in a thread;
            [2,3] //arguments for the function;
           ).then(function (data) { 
                     var foo = data; 
                     console.log(foo);
           });

vkThread is built on HTML5 "Worker" technology. It also incorporates JSONfn code to implement key tasks. External "when()" or jQuery library are optional.

Plugin is very compact. Development version (plain text with comments) is less than 2.5k.
