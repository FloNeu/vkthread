Overview

vkThread is a javascript plugin, which allows you to execute a piece of your code in a thread.

Plugin is built on HTML5 "Web Workers" technology and uses ES6 "Promise" object.*

Almost any type of function can be opened in a thread with this plugin:

  • Regular functions
  • Object's methods
  • Functions which dependents on library
  • Anonymous functions
  • Lambda functions

Basic usage:

Install with bower


bower install vkthread

integrate plugun in your project


<script src="../vkthread/vkthread.min.js" type="text/javascript"></script>

create function

function foo(num1, num2) {
	return num1 + num2;
}

execute this function in a thread:

var param = {
    fn: foo,     // <-- function to run in a thread;
    args: [2, 3] // <-- function's arguments;
};

vkthread.exec(param).then(
    function(data){
        console.log(data); // <-- prints 5;
    }
);
